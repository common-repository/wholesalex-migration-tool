## Migration Available For
We have a plan for making the migration tool available for all WooCommerce B2B store owners, no matter what wholesale plugins they are using. After getting feedback and requests from the potential WholesaleX user, we have made it available for the following plugins:

* Wholesale Suite [<a href="https://getwholesalex.com/docs/wholesalex/wholesalex-migration-tool/migrate-from-b2bking/" target="_blank">Documentation Link/a>]
* B2BKing [<a href="https://getwholesalex.com/docs/wholesalex/wholesalex-migration-tool/migrate-from-wholesale-suite/" target="_blank">Documentation Link</a>]

However, we are in a continuous development process to make the migration available for the other WooCommerce Wholesale solutions.

## Prerequisite 

If you want to migrate your existing WooCommerce B2B store data, you must have the following plugins.

* WholesaleX - Migration Tool
* WholsaleX - WooCommerce B2B Solution
* Wholesale Suite - (If you are using it to build and manage your WooCommerce Wholesale store.)
* B2BKing - (If you are using it to build and manage your WooCommerce Wholesale store.)

## Getting Started with WholesaleX - Migration Tool

Once you have installed WholesaleX and the Migration plugin, you will see a new option as the Migration tool has been added with other WholesaleX options. And once you click on the migration option, you will see options (Wholesale Suite and B2BKing).

Select the plugin that you are using for creating and managing your Wholesale store. Then, you just need to click on the Migration Now button. Once you click on the button, the migration will be started, and you will see a confirmation message once it's completed.

== Installation ==

= Minimum Requirements =

* WordPress 5.0.0 or greater
* PHP version 7.3 or greater
* MySQL version 5.6 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don‚Äôt even need to leave your web browser. 

1. Go to your WordPress Plugin installation menu (Dashboard > Plugins > Add New)
2. In the search field type "WholesaleX - Migration Tool" and press enter.
3. "Install Now" and then click "Active"


= Manual installation =

For Manual installation, you download our product from WordPress directory uploading it to your web-server via your FTP or CPanel application.

1. Download the plugin and unzip it
2. Using an FTP program or CPanel upload the unzipped plugin folder to your WordPress installation‚Äôs wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu (Dashboard > Plugins > Installed Plugins) within the WordPress admin.

= Upgrading =

Automatic update working great for this plugin. If you customize the plugin make sure to store a backup before update.

== Frequently Asked Questions ==

= Does it work with any plugins? =
No, it doesn't work with all B2B plugins. Currently, it works with only Wholesale Suite and B2BKing. However, it will bring support to more plugins in the future.
= Does this plugin offer updates and support? =
Yes, this plugin will be updated on a regular basis, and you will get dedicated support for both WholesaleX and the migration plugin.
= Do I need coding knowledge to use this plugin? =
No, you don't need to have any kind of coding knowledge to use this plugin.
